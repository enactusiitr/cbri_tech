# Tech Stack for `cbri_tech`

## 1. Frontend (Web Application)
- **Framework**: Next.js (`16.0.10`), React (`19.2.0`)
- **Styling**: Tailwind CSS (`4.2.2`)
- **UI Components**: Radix UI
- **Forms & Validation**: React Hook Form, Zod
- **Icons**: Lucide React
- **Data Visualization**: Recharts
- **Other Utilities**: date-fns (date formatting), embla-carousel-react (carousels), sonner (toast notifications)

## 2. Backend (REST API Server)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database ORM**: Mongoose (MongoDB)
- **Real-time Communication**: Socket.io
- **Utilities**: dotenv (environment variables), cors

## 3. Mobile App (Restaurant Dashboard)
- **Framework**: Flutter
- **State Management**: Provider
- **Networking/API**: http, socket_io_client (real-time updates)
- **Local Storage/Config**: flutter_dotenv
- **UI/UX Enhancements**: Google Fonts, Shimmer (loading skeletons), Audioplayers (custom sounds)
- **Notifications**: flutter_local_notifications
